## Instalation

1.  Copy the xml files to `~/.gnome2/gedit/styles/`
2.  In GEdit: Edit > Preferences > Font & Colors
3.  Click Add
4.  Browse to `~/.gnome2/gedit/styles/`
5.  Select the styles files
